Livecountsedit changelogs

All dates listed below are in Pacific Time (UTC-8/UTC-7).

6.10.5 - June 15, 2024

Link: https://lcedit.com/studio/

Github commit: https://github.com/livecountsedit/livecountsedit.github.io/commit/bb2d18e25c3ad705316e733a0ab29624f45642ac

Changes

  • Fixed the YouTube Studio animation and added the new odometer library
    • Leading zeros no longer show during the animation if the number of digits change (as in the MrBeast 100 million livestream).
    Cleaned up the code by using the new odometer
    This also means negative numbers and numbers with more than 12 digits work now. Just know that if you go above 16 digits, the last few digits may not work correctly, and going above 21 digits can cause things to break (this is due to how floating point numbers work)
    The new odometer library will soon be added to other counters and allow you to customize the odometer animation.

    Fixes

  • Huge thanks to Weiyu for helping fix a few things on other counters!
    • Avatar on right side of compare mode on Akshatmittal was not working
    YouTube icon was missing on SocialBlade

    6.10.4 - May 12, 2024

    Link: https://lcedit.com/top50/

    Github commit: https://github.com/livecountsedit/livecountsedit.github.io/commit/f6632634a9b4ef362f7e0497e31d0ee7ce54f633

    Changes

  • Made the styles of top 50 more "enjoyable" (dark theme)
  • Added more options for customizing the FYSC (youtube integration)

  • Started work on an animated cards option
    • This is not easy as I've learned the hard way many times...

    • 6.10.1 - April 7, 2024

    Link: https://lcedit.com/top50/

    • 6.10.0 - April 6, 2024

    Link: https://lcedit.com/top50/

    Github commit: https://github.com/livecountsedit/livecountsedit.github.io/commit/1943a4ebaa01e0f60e25d010d2af1638f0a6acfd

    Changes

  • Added differences (sub gaps) to Top 50.
    • Check the "Show Differences" option to enable.
  • Added a "Zero all counters" button to Top 50.
    • This button will set all counters to zero after a confirmation.
  • Added a "Show Blank Slots" option to Top 50.
    • If unchecked, only the box for the slot will appear in blank slots.
  • Blank slots are now shown with as a blank channel with a blank profile picture instead of the default profile picture and "Loading".

    • Fixes

  • HTML injection issues
  • Fastest icon not showing up when slowest icon is enabled and the gains for all channels are tied
  • Overflow pushing the counter below the image so it is no longer visible
